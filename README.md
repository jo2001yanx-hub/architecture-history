# 🏛️ Architecture History Learning Platform

An interactive, visual timeline for learning architectural history through famous precedent projects.

## ✨ Features

- **📅 Chronological Timeline** — Browse from Ancient to Contemporary periods
- **🏢 Precedent Database** — 17+ landmark buildings with architectural analysis
- **🔍 Comparison Tool** — Analyze 2 buildings side-by-side across different eras
- **👥 Community-Driven** — Contribute new buildings and expand the database
- **⚡ Fully Offline** — No servers, all runs in your browser

## 🚀 Quick Start

1. Clone this repository: `git clone https://github.com/yourusername/architecture-history.git`
2. Open `index.html` in your web browser
3. Start exploring the timeline!
